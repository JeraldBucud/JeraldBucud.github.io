FEATURED SYSTEM 01
Full-stack system Private repository In active development

Godscent Drive POS

A point-of-sale and inventory-management system designed for businesses that sell products across warehouses and temporary event locations. The system manages products, suppliers, stock transfers, event inventory, sales, refunds, promotions, reports and audit records.

Problem

Maintain accurate sales and inventory across permanent and temporary locations, including events where internet access may be weak or unavailable.

Approach

Spring Boot REST services, React workflows, PostgreSQL records and IndexedDB queues support online and offline selling.

  • Java 21
  • Spring Boot

  • IndexedDB
